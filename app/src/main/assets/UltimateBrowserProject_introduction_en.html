UltimateBrowserProject v.1.0-ß.

UltimateBrowserProjectBeta is an alternative to the original application, providing new features before everyone else gets them. These functions may be buggy, and users are expected to report the bugs.

Fullscreen Mode:

  • You can now enable Fullscreen mode in the application settings.

File Handling:

UBP Beta now handles HTML Files.

Restore Previous Sessions:

  • You can now Restore tabs from previous sessions by changing the settings at Setting/Restore Tabs.

Deep Linking:

  1. You can now open links in the application itself.

  2. This requires you to have the application installed on your phone

Omnibox Scrolling

Omnibox now scrolls with the webpage by default!

Log Reporting:

UltimateBrowserProjectBeta can automatically capture Log Reports when it crashes.

When the crash takes place, an Alert Dialog will pop up requesting you to send the report. All you have to do is add comments regarding the problem (OPTIONAL), and click OK.

Everything else will be handled by the application itself!

Although you can report bugs on the Application Thread manually.
